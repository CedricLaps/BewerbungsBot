# Beispiel-Anschreiben (Stilvorlagen)

Lege hier frühere Anschreiben von dir ab — die KI nutzt sie als Stilvorlage,
damit generierte Anschreiben so klingen wie du.

- Unterstützte Formate: `.txt`, `.md`, `.pdf`
- Es werden bis zu **3 Dateien** verwendet (alphabetisch sortiert),
  jede gekürzt auf die ersten ~1500 Zeichen
- Die KI übernimmt Tonfall und Formulierungsstil, kopiert aber keine Sätze
  wörtlich und übernimmt keine Firmen-/Stellendetails aus den Beispielen
- Diese README wird ignoriert

Änderungen wirken sofort beim nächsten generierten Anschreiben —
kein Neustart nötig (der Ordner ist live in den Container eingebunden).
